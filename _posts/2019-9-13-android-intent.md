# Tipos de Intent
## Interno
Toda activity que consulta um recurso externo.
Exemplos:
- Maps
- Feed Rss


## Externo
Toda activity que necessita de uma interação do usuário.
Exemplos:
- Item de uma List/Recycler View
- Transição de Login para Main Activity
